Contents

##box-sizing

由于各种浏览器有各自的盒模型，有自己的标准，所以如果设置margin,padding他们所加在的位置也会不同，这样看到的宽高可能就会不一样了。这样我们有box-sizing这个属性可以设置，它有两个值content-box和border-box，如果box-sizing: content-box，那么元素设置宽高后，加padding,margin，还是会按照浏览器标准的不同而解析；设置box-sizing: border-box 元素宽高在各种浏览器则会一致，不会因padding,border而改变。
一些专家甚至建议所有的WEB开发者：将所有的元素的box-sizing都设为border-box

##flex

如果不知道元素高度，那么垂直居中就不容易实现，所以我们可以display: flex，然后设置align-items: center就可以了，其他几个属性：
flex-direction: row|row-reverse|column|column-reverse 设置元素的排列顺序
flex-wrap: nowrap|wrap|wrap-reverse  设置一行排不下如何换行
justify-content: flex-start|flex-end|center|space-between|space-around  x轴的对齐方式
align-items: flex-start|flex-end|center|baseline|stretch  y轴的对齐方式
